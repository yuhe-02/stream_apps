このファイルの使い方

  1. 仮想環境を準備します。
    • 私の場合は、"virtualenv"を使用しました
  2. 仮想環境を立ち上げ、環境上で、pip install streamlit を入力する
  3. streamlit helloと入力し、ページが遷移したら成功です。
  4. input_form.pyの編集を行います。API先がStability AIのため、APIキーを取得し該当部分を編集します。そのためにアカウント作成から行っていきます。
    1. Stability AIでアカウントを作成します。
    2. ユーザーページを開き、アカウントからAPIキーを取得します。
    3. input_form.pyの中にある該当部分にペーストします。
  5. ターミナルでstreamlit run input_form.pyと入力し、入力画面に遷移したら成功です。
